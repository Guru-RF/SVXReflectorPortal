Desktop

macOS

Your system
Apple Silicon M1–M4 Intel Macs before ~2020
How to install
  1. Open the .dmg and drag SVX Portal into Applications.
  2. Launch it from Applications — the apps are signed & notarized by Apple, so they open normally.
Windows
Installer · .exe
Your system
64-bit (x64) Most PCs ARM64 Surface / ARM PCs
“Windows protected your PC”? (unsigned app)
  1. Download and double-click the Setup .exe.
  2. A blue “Windows protected your PC” (SmartScreen) box appears — this is normal, the app just isn’t code-signed.
  3. Click the small More info link.
  4. Click the Run anyway button that appears.
  5. Continue through the installer as usual.
Linux
Portable · .AppImage
Your system
x86_64 Most PCs ARM64 Raspberry Pi / ARM
How to run an AppImage
  1. Download the .AppImage file.
  2. Make it executable — right-click → Properties → Permissions → tick Allow executing as program, or in a terminal: chmod +x SVX-Portal-x86_64.AppImage
  3. Double-click it, or run ./SVX-Portal-x86_64.AppImage
